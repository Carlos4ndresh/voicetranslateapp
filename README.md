# Voice Translator App

This is a demo repository of the capabilities of some AWS Services: 

    - S3
    - Lambda
    - Transcribe 
    - Translate
    - Polly
    - CloudFront

Is a small online translator that takes as input a in-browser voice recording, then it transcribes it, translate it and syntethize the desired translation outcome.

This repo is forked of: https://github.com/tstachlewski/voice-translator-app



Note: This repo fixed Frontend bugs. The intention is to enhance this demo with additional capabilities
